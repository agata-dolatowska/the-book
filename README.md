# the-book

## Install

install [node.js](https://nodejs.org/en/download/)

```
git clone https://github.com/agata-dolatowska/the-book
cd the-book
npm install
npm start
```

get [bible api key](https://scripture.api.bible/)
create config.ts file in project folder

```
export const config = {
  bibleApiKey: "your key"
};
```
